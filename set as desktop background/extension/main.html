This extension is compatible with Windows and requires the installation of Python and Script, which serve as the native application and messaging host needed to make changes to your computer

Instructions

To change your wallpaper, right click an image on the web and select Set as desktop background

To save the most recently applied wallpaper to your downloads, click on the toolbar button

Notifications are used to display error messages and available script updates
